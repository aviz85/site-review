Features

Everything wired in.
Nothing to configure.

SiteReel orchestrates six different AI tools — all called automatically from a single skill invocation.

🎥

GIF Recording

Chrome browser automation records a real exploration session as an animated GIF. No screen recorder needed — built into the skill.

🎙️

ElevenLabs Voice

Cloned voice narration via ElevenLabs multilingual v2. Consistent, professional audio with direction cues baked into the script.

🎵

AI Music

Custom background score generated per review. Corporate ambient at 30% volume — present, audible, never intrusive. Never the same track twice.

🔁

Smart Looping

GIF duration is measured and automatically looped to match narration length. Narration sets the pace — video follows.

💬

Auto Subtitles

Groq Whisper word-level transcription generates SRT files. Subtitles are reviewed, corrected, and burned directly into the video.

↔️

RTL Support

Hebrew and Arabic subtitle content is automatically detected and wrapped with Unicode directional marks before embedding. No configuration needed.

Get started

One skill.
Any platform.

SiteReel is a Claude Code skill — install it once, use it everywhere. No configuration, no API setup. Just point it at any URL. 

# Skill location
~/.claude/skills/site-review/

# Usage — English review
/site-review https://yourapp.com

# Hebrew narration with RTL subtitles
/site-review https://yourapp.com --lang he

# Reuse existing GIF, skip recording
/site-review https://yourapp.com --gif /tmp/my-session.gif

# Custom output directory
/site-review https://yourapp.com --output ~/reviews/myapp
